## 套路库

<a name="套路库"></a>

### 哈希表
**何时用**: 需要快速查找、计数、去重
**模板**:
```python
counter = {}
for x in nums:
    counter[x] = counter.get(x, 0) + 1
```
**关键**: 空间换时间，O(1) 查找

### 双指针
**何时用**: 数组左右夹逼、链表找中点
**模板**:
```python
left, right = 0, len(nums) - 1
while left < right:
    # 判断并移动
    if condition:
        left += 1
    else:
        right -= 1
```
**关键**: 有序数组、左右夹逼

### 滑动窗口
**何时用**: 子串/子数组问题
**模板**:
```python
left = 0
for right in range(len(s)):
    # 扩大窗口
    while valid(right - left + 1):
        # 收缩窗口
        left += 1
    # 更新答案
```
**关键**: 扩张+收缩，维护窗口

### 单调栈
**何时用**: 下一个更大/更小/最大矩形
**模板**:
```python
stack = []  # 存索引
for i in range(n):
    while stack and heights[i] < heights[stack[-1]]:
        h = heights[stack.pop()]
        w = i - stack[-1] - 1 if stack else i
        area = h * w
    stack.append(i)
```
**关键**: 递减栈找边界

### 二分查找
**何时用**: 有序数组找目标
**模板**:
```python
left, right = 0, len(nums) - 1
while left < right:
    mid = (left + right) // 2
    if condition:
        left = mid + 1
    else:
        right = mid
return left
```
**关键**: 收缩边界，夹逼目标

### DFS/BFS
**何时用**: 树/图遍历、连通性
**模板（DFS）**:
```python
def dfs(node, visited):
    if not node or node in visited:
        return
    visited.add(node)
    process(node)
    for neighbor in node.neighbors:
        dfs(neighbor, visited)
```
**关键**: 状态标记，防止重复

### DP
**何时用**: 最优子结构、重叠子问题
**模板**:
```python
dp = [0] * (n + 1)
for i in range(1, n + 1):
    dp[i] = min(dp[i], dp[i-coin] + 1)  # 转移方程
```
**关键**: 状态定义 + 转移方程

### 回溯
**何时用**: 组合、排列、子集
**模板**:
```python
def backtrack(path, choices):
    if valid(path):
        result.append(path[:])
        return
    for choice in choices:
        path.append(choice)
        backtrack(path, choices)
        path.pop()  # 撤销
```
**关键**: 选择 + 递归 + 撤销
